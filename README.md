# toi_2
ข้อสอบทดสอบการแข่งขันคอมพิวเตอร์โอลิมปิกระดับชาติ ครั้งที่ 2
(มหาวิทยาลัยบูรพา)

ประกอบไปด้วยโจทย์ :
- ผึ้งสำยพันธุ์พิเศษ
- ลำดับย่อยของตัวเลขที่มีตำแหน่งต่อเนื่องกันที่มีค่ำสูงสุด
- ลูกเต๋ำ
- ระบบแสดงผลตัวเลขแบบเจ็ดส่วน
- เส้นขอบฟ้า
- ปริศนาค้นหาคา
